If your Claude tooling supports a custom base URL and API key, you can point it at the AI Stats Gateway and continue using `/v1/messages`.

## Recommended setup

- Base URL: `https://api.phaseo.app/v1`
- Auth: `Authorization: Bearer <AI_STATS_API_KEY>`
- Endpoint: `/messages`

## Tips

- Verify model ids with the [Models endpoint](../../api-reference/endpoint/models.mdx).
- If you need tool calling or structured outputs, confirm model support first.
