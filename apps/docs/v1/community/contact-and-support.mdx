## Report a bug

Open an issue in GitHub with clear reproduction steps and any relevant logs:

- https://github.com/AI-Stats/AI-Stats/issues

## Ask for help

Join the Discord community for questions and support:

- https://discord.gg/aQyywCvgZ5

## Provide feedback

Feedback is welcome in GitHub issues or Discord. If you are unsure where to post, start with GitHub issues and the team will route it.
