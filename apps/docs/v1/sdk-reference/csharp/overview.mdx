---
title: "C# SDK (preview)"
description: "Local-preview C# wrapper for the AI Stats API."
---

<Note type="warning">
  The C# SDK is in preview and is not yet published to NuGet.
</Note>

The local wrapper at `packages/sdk/sdk-csharp/Client.cs` currently exposes these methods:

- `GenerateText(...)`
- `GenerateResponse(...)`
- `GenerateImage(...)`
- `GenerateModeration(...)`
- `ListModels(...)`
- `Health()`

```csharp
using AiStatsSdk;

var client = new Client("YOUR_KEY");
var models = await client.ListModels(new Dictionary<string, string> { ["limit"] = "5" });
```
