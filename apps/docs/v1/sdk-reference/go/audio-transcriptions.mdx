---
title: "Audio transcriptions"
description: "Use the generated Go client for /audio/transcriptions."
---

The preview Go wrapper does not yet expose an audio-transcription helper. Use
the generated client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

resp, err := gen.CreateTranscription(client, nil, nil, nil, map[string]interface{}{
  "model": "openai/whisper-1",
  "audio_b64": "data:audio/wav;base64,...",
})
if err != nil {
  panic(err)
}
fmt.Println(resp)
```
