---
title: "Video generation"
description: "Use the generated Go client for /videos."
---

The preview Go wrapper does not yet expose video helpers. Use the generated
client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

job, err := gen.CreateVideo(client, nil, nil, nil, map[string]interface{}{
  "model": "openai/sora-2",
  "prompt": "A cinematic sunrise over a mountain lake",
})
if err != nil {
  panic(err)
}

fmt.Println(job)
```
