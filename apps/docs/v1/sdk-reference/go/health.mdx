---
title: "Health"
description: "Use the generated Go client for /health."
---

The preview Go wrapper does not yet expose a health helper. Use the generated
client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

health, err := gen.Healthz(client, nil, nil, nil, nil)
if err != nil {
  panic(err)
}

fmt.Println(health)
```
