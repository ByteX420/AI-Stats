---
title: "Audio speech"
description: "Use the generated Go client for /audio/speech."
---

The preview Go wrapper does not yet expose an audio-speech helper. Use the generated
client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

resp, err := gen.CreateSpeech(client, nil, nil, nil, map[string]interface{}{
  "model": "openai/gpt-4o-mini-tts",
  "input": "Hello from AI Stats",
})
if err != nil {
  panic(err)
}
fmt.Println(resp)
```
