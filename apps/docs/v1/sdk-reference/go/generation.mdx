---
title: "Generation lookup"
description: "Use the generated Go client for /generations."
---

The preview Go wrapper does not yet expose generation-lookup helpers. Use the
generated client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

record, err := gen.GetGeneration(client, nil, map[string]string{
  "id": "G-01ABC...",
}, nil, nil)
if err != nil {
  panic(err)
}

fmt.Println(record)
```
