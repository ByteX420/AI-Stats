---
title: "Image edits"
description: "Use the generated Go client for /images/edits."
---

The preview Go wrapper does not yet expose image-edit helpers. Use the
generated client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

resp, err := gen.CreateImageEdit(client, nil, nil, nil, map[string]interface{}{
  "model": "openai/gpt-image-1",
  "prompt": "Turn this into a watercolor style",
  "image": "data:image/png;base64,...",
})
if err != nil {
  panic(err)
}

fmt.Println(resp)
```
