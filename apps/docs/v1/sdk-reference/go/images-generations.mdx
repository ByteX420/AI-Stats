---
title: "Image generations"
description: "Use the generated Go client for /images/generations."
---

The preview Go wrapper does not yet expose image-generation helpers. Use the
generated client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

resp, err := gen.CreateImage(client, nil, nil, nil, map[string]interface{}{
  "model": "openai/gpt-image-1",
  "prompt": "A lighthouse at sunset",
})
if err != nil {
  panic(err)
}

fmt.Println(resp)
```
