---
title: "Files"
description: "Use the generated Go client for /files."
---

The preview Go wrapper does not yet expose file helpers. Use the generated
client in `src/gen`.

```go
import (
  "fmt"
  gen "github.com/AI-Stats/ai-stats-go-sdk-wrapper/src/gen"
)

client := gen.NewClient("https://api.phaseo.app/v1")
client.Headers["Authorization"] = "Bearer " + apiKey

files, err := gen.ListFiles(client, nil, nil, nil, nil)
if err != nil {
  panic(err)
}

fmt.Println(files)
```
