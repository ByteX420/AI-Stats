<Note type="warning">
  `upload_file`, `list_files`, and `get_file` currently return `coming_soon` placeholders.
</Note>

```python
uploaded = client.upload_file(purpose="batch", file="...")
files = client.list_files()
file_obj = client.get_file("file_123")

print(uploaded)
print(files)
print(file_obj)

#   "status": "coming_soon",
```
