## Local build

- Requires a recent Java JDK and Maven.
- From the repo root, go to `packages/sdk/sdk-java`.
- Build with Maven and use the jar from `target/` in your project.

```bash
cd packages/sdk/sdk-java
mvn -DskipTests package
```
